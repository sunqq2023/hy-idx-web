# 如何快速查询空投代币的交易对

## 🎯 快速方法（推荐）

### 方法 1: 使用 BSCScan（最快最准确）

1. **打开 BSCScan**

   ```
   https://bscscan.com/
   ```

2. **搜索代币地址**
   - 在 TP 钱包中复制代币合约地址
   - 粘贴到 BSCScan 搜索框
   - 点击搜索

3. **查看交易对信息**
   - 在代币页面，点击 "DEX Trades" 标签
   - 可以看到所有交易对和交易所
   - 显示流动性、交易量等信息

4. **直接跳转到 PancakeSwap**
   - 在代币页面，点击 "DEX" 按钮
   - 选择 "PancakeSwap"
   - 自动跳转到交易页面

**示例**：

```
代币地址: 0xc98F60B3F98E8Bf860436746db637e13B0e17458
BSCScan: https://bscscan.com/token/0xc98F60B3F98E8Bf860436746db637e13B0e17458
```

---

### 方法 2: 使用 DexScreener（最直观）

1. **打开 DexScreener**

   ```
   https://dexscreener.com/
   ```

2. **搜索代币地址**
   - 粘贴代币合约地址
   - 自动显示所有交易对

3. **查看详细信息**
   - 价格图表
   - 流动性
   - 交易量
   - 所有交易对列表
   - 支持的 DEX

4. **直接交易**
   - 点击 "Trade" 按钮
   - 跳转到对应的 DEX

**优点**：

- ✅ 界面直观，图表清晰
- ✅ 显示所有 DEX 的交易对
- ✅ 实时价格更新
- ✅ 可以看到流动性分布

---

### 方法 3: 使用 PancakeSwap 直接搜索

1. **打开 PancakeSwap**

   ```
   https://pancakeswap.finance/swap
   ```

2. **选择代币**
   - 点击 "Select a currency"
   - 粘贴代币合约地址
   - 如果有交易对，会自动显示

3. **查看交易对**
   - 如果代币存在，会显示代币信息
   - 可以看到价格和流动性
   - 如果没有交易对，会显示 "No results found"

**注意**：

- ⚠️ 只能看到 PancakeSwap 上的交易对
- ⚠️ 如果代币在其他 DEX 上，这里看不到

### 方法 4: 使用命令行查询（技术向）

```bash
#!/bin/bash

TOKEN_ADDRESS="0xYourTokenAddress"
FACTORY="0xcA143Ce32Fe78f1f7019d7d551a6402fC5350c73"  # PancakeSwap Factory
WBNB="0xbb4cdb9cbd36b01bd1cbaebf2de08d9173bc095c"
USDT="0x55d398326f99059ff775485246999027b3197955"
BUSD="0xe9e7cea3dedca5984780bafc599bd69add087d56"

echo "检查 PancakeSwap 交易对..."

# 检查 TOKEN/WBNB
PAIR_BNB=$(cast call $FACTORY "getPair(address,address)(address)" $TOKEN_ADDRESS $WBNB --rpc-url https://bsc-dataseed.binance.org/)
echo "TOKEN/WBNB: $PAIR_BNB"

# 检查 TOKEN/USDT
PAIR_USDT=$(cast call $FACTORY "getPair(address,address)(address)" $TOKEN_ADDRESS $USDT --rpc-url https://bsc-dataseed.binance.org/)
echo "TOKEN/USDT: $PAIR_USDT"

# 检查 TOKEN/BUSD
PAIR_BUSD=$(cast call $FACTORY "getPair(address,address)(address)" $TOKEN_ADDRESS $BUSD --rpc-url https://bsc-dataseed.binance.org/)
echo "TOKEN/BUSD: $PAIR_BUSD"
```

## 🔍 详细步骤示例

### 完整流程：从 TP 钱包到交易

#### 步骤 1: 获取代币地址

1. 打开 TP 钱包
2. 找到收到的空投代币
3. 点击代币
4. 点击 "合约地址" 或 "详情"
5. 复制合约地址

#### 步骤 2: 在 BSCScan 查询

1. 打开 https://bscscan.com/
2. 粘贴代币地址到搜索框
3. 查看代币信息页面

**关键信息**：

- ✅ 代币名称和符号
- ✅ 总供应量
- ✅ 持有人数量
- ✅ 转账次数
- ✅ DEX 交易对

#### 步骤 3: 查看交易对

在 BSCScan 代币页面：

1. **点击 "DEX Trades" 标签**
   - 显示所有 DEX 上的交易对
   - 可以看到：
     - 交易对（如 TOKEN/USDT）
     - 交易所（如 PancakeSwap V2）
     - 流动性
     - 24h 交易量

2. **点击 "Holders" 标签**
   - 查看流动性池地址
   - 流动性池通常是前几名持有者

3. **点击 "Info" 标签**
   - 查看代币基本信息
   - 是否验证合约
   - 是否有官网

#### 步骤 4: 跳转到交易所

**方法 A: 通过 BSCScan**

- 在代币页面点击 "DEX" 按钮
- 选择 PancakeSwap 或其他 DEX
- 自动跳转到交易页面

**方法 B: 手动打开 PancakeSwap**

- 打开 https://pancakeswap.finance/swap
- 粘贴代币地址
- 选择交易对（如 USDT、BNB）

## ⚠️ 安全检查（重要！）

在交易空投代币前，务必检查：

### 1. 代币是否是骗局

**检查项**：

- ❌ 合约是否开源验证？
- ❌ 持有人数量是否正常？（少于 100 人要小心）
- ❌ 流动性是否充足？（少于 $1000 要小心）
- ❌ 是否有大量转账记录？
- ❌ 是否有官网和社交媒体？

**危险信号**：

- 🚨 合约未验证
- 🚨 持有人很少
- 🚨 流动性极低
- 🚨 无法卖出（蜜罐合约）
- 🚨 高额税费（买卖税超过 10%）

### 2. 使用蜜罐检测工具

**推荐工具**：

1. **Honeypot.is**

   ```
   https://honeypot.is/
   ```

   - 粘贴代币地址
   - 自动检测是否是蜜罐
   - 显示买卖税费

2. **Token Sniffer**

   ```
   https://tokensniffer.com/
   ```

   - 代币安全评分
   - 检测恶意代码
   - 显示风险等级

3. **RugDoc**
   ```
   https://rugdoc.io/
   ```

   - 项目审计
   - 风险评估

### 3. 小额测试

**建议流程**：

1. 先用极小金额测试（如 $1）
2. 确认可以卖出
3. 检查实际到手金额
4. 计算买卖税费
5. 如果正常，再考虑大额交易

## 📊 常见交易对

### BSC 主网常见交易对

| 交易对     | 说明     | 流动性   |
| ---------- | -------- | -------- |
| TOKEN/WBNB | 最常见   | 通常最高 |
| TOKEN/USDT | 稳定币对 | 较高     |
| TOKEN/BUSD | 稳定币对 | 中等     |
| TOKEN/ETH  | 跨链代币 | 较低     |

### 如何选择交易对

1. **优先选择流动性高的**
   - 滑点更低
   - 价格更稳定
   - 交易更快

2. **考虑交易成本**
   - WBNB 对：需要 BNB 做 gas
   - USDT 对：价格更稳定
   - 直接对：可能滑点更高

3. **查看价格差异**
   - 不同交易对价格可能不同
   - 选择价格最优的

## ❓ 常见问题

### Q1: 为什么找不到交易对？

**可能原因**：

1. 代币刚发行，还没有流动性
2. 代币是骗局，没有真实交易对
3. 代币只在小型 DEX 上有交易对
4. 代币地址输入错误

**解决方法**：

- 在 DexScreener 搜索（覆盖所有 DEX）
- 检查代币是否真实
- 等待一段时间再查询

### Q2: 交易对显示但无法交易？

**可能原因**：

1. 蜜罐合约（只能买不能卖）
2. 流动性被移除
3. 交易暂停
4. 高额税费

**解决方法**：

- 使用 Honeypot.is 检测
- 查看流动性是否充足
- 小额测试

### Q3: 不同交易对价格不一样？

**原因**：

- 流动性分散在不同池子
- 套利机器人还没平衡价格
- 某些池子流动性太低

**建议**：

- 选择流动性最高的交易对
- 注意滑点设置
- 避免在流动性低的池子交易

### Q4: 如何判断空投是否值得卖？

**检查清单**：

- ✅ 代币有真实项目支持
- ✅ 流动性充足（>$10,000）
- ✅ 持有人数量合理（>1000）
- ✅ 有交易量
- ✅ 合约已验证
- ✅ 通过蜜罐检测

**如果以上都满足**：

- 可以考虑持有或卖出
- 建议先卖出一部分回本

**如果不满足**：

- 很可能是骗局
- 建议直接忽略或小额测试
